Anup Chemistry is an educational website designed for Class 11, Class 12, and JEE (Main) students.
This website provides complete information about batches, courses, results, and enrollment process in a clean and responsive UI.

Features
Information about Arjuna, Lakshya & Yakeen batches
Chemistry focused coaching for 11th, 12th & JEE
Fully Responsive Design
Modern UI using Bootstrap
Dummy Payment / Enrollment Page
Carousel, cards & clean layout
Fast loading static website
